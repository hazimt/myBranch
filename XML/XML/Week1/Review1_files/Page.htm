This page is an Overview Page for Week 1 Module. It is used in this learning module to guide you through the learning activities. 

Goal                                                  
Get ready for the first class meeting: get supplies, software, and text book ready for the class. Overview the course syllabus. Create first XML document.
Objectives
The students will listen to the class presentation and participate in the hands-on exercise to achieve the following objectives:
·         Understand XML vocabularies
·         Define well-formed and valid XML
·         Create an XML declaration, elements and attributes
·         Work with comments, character and entity references
·         Describe how XML handles character data, parsed character data, and white space.
·         Work with XML parsers, specifically built into browsers
·         Apply a style sheet to an XML document
